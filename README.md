# Network Traffic Analysis Using Wireshark

## Objective
To capture and analyze network traffic to identify normal and suspicious activity.

## Tools Used
- Wireshark
- Kali Linux

## Analysis Performed
- TCP Handshake Analysis
- HTTP Traffic Inspection
- DNS Query Analysis
- Port Scan Detection

## Conclusion
Network traffic analysis is essential for detecting anomalies and improving network security.

## Disclaimer
This project was conducted for educational purposes in a controlled environment.
